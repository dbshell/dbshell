DB Shell Samples

Hello world

It can be used for basic checking whether DB Shell is working (no database is needed). In Samples folder, run in command line: 

..\bin\DbShell HelloWorld.dbsh

Configure connection and create testing database

  • In Samples folder, open file Configuration.dbsh
  • Edit connection to your Microsoft SQL server
  • You can also change database name
  • Run script CreateDatabase.dbsh using comamnd line: ..\bin\DbShell CreateDatabase.dbsh. This script is heavily using Script tag.
If you can see message "Your samples database was succesfully created", you are able to run other samples working with database.
