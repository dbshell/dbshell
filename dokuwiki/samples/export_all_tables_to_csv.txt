====== Export all tables to CSV ======

This example exports all tables in database into CSV files. Each table is exported into separated file, its name is TABLE.csv (TABLE is replaced with real table name).

<code xml>
<?xml version="1.0" encoding="utf-8" ?>
<ForEach 
  xmlns="http://schemas.dbshell.com/core"
  Source="{Tables}"
  Property="Table"
>
  <CopyTable Source="{Table ${Table.Name}}" Target="{File Export/${Table.Name}.csv}" />
</ForEach>
</code>
