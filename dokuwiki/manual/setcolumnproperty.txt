====== SetColumnProperty element ======
Assigns property of column. It is used mainly for generating database documentation. Could be used fo for batch assign - all columns, which meets conditions, are influenced.

  * Attribute **Name**: mame of property
  * Attribute **Table**: name of table
  * Attribute **Tables**: regular expression, matching againts table name
  * Attribute **Column**: name of column
  * Attribute **Columns**: regular expression, matching againts column name
  * Attribute **Value**: value, which will be set
  * Attribute **IsPrimaryKey**: if ''true'', only primary key columns are processed (default: ''false'')
  * Attribute **IsIdentity**: if ''true'', only identity columns are processed (default: ''false'')
