====== CsvFile element ======

Provides read/write access to CSV file. Allows to specify more details than use of generic [[File]] element.

  * Implements: **ITabularDataSource**, **ITabularDataTarget**
  * Attribute **Name**: file name
  * Attribute **Delimiter**: column delimiter (by defaukt ',' - comma)
  * Attribute **Quote**: quitong character (by default '"')
  * Attribute **Escape**: string escape character (by default '"'). Used when **Quote** is find inside stored text
  * Attribute **Comment**: prefix of comment lines (by default '#'). Used only when reading
  * Attribute **HasHeaders**: ''true'' if CSV file has header row with columns names. Default: ''true''
  * Attribute **QuotingMode**: one of
    * Always - all fields are quoted
    * AlwaysExceptNumbers - all fields except numbers are quoted
    * OnlyIfNecessary - quoting markrks are used only of necessary
  * Attribute **Encoding** - file encoding, by default UTF-8
  * Attribute **TrimSpaces**: ''true'' of trim spaces of inout file. default: ''false''
