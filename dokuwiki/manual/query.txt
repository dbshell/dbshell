====== Query element ======

Query element providers acces to result of query execution.

  * Implements: **ITabularDataSource**, **IListProvider**, **IEnumerable**, **IModelProvider**
  * Attribute **Text**: query text (could be written also directly as as content)
  * Extesion syntax: ''Property="{Query 'select * from Album'}"''
