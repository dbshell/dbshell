====== SaveToFile element ======

Writes string or binary data to file.

  * Implements: **IRunnable**
  * Property **File**: output file name
  * Property **Value**: The value to be saved. Can contain replace expressions. If set, Expression cannot be set. Can be used only for textual data. When saving binary data, use Expression instead.
  * Property **Expression**: The expression to be saved. If set, Value cannot be set.
  * Property **Encoding**: Output file encoding (default: ''UTF-8'')
