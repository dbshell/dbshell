====== Razor element ======

Razor element runs template in [[http://www.w3schools.com/aspnet/razor_syntax.asp|razor syntax]]. 

  * Implements: **IRunnable**
  * Attribute **File**: output file name
  * Attribute **Model**: model ofr razor template. If model is of type string, it is evaluated as expression
  * Attribute **TemplateFile**: file with razor template
  * Attribute **TemplateData**: contains directly razor template (can be written directly as element content)
