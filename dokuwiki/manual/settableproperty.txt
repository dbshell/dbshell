====== SetTableProperty element ======
Assigns property of table. It is used mainly for generating database documentation. Could be used fo for batch assign - all tables, which meets conditions, are influenced.

  * Attribute **Name**: mame of property
  * Attribute **Table**: name of table
  * Attribute **Tables**: regular expression, matching againts table name
