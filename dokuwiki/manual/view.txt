====== View element ======

View element enapsulates view in database. It could be used in following contexts:
  * Data source - Source property of [[CopyTable]] element
  * List of rows - model for [[Razor]] element


  * Implements: **ITabularDataSource**, **IListProvider**, **IEnumerable**, **IModelProvider**
  * Attribute **Name**: view name
  * Attribute **Schema**: view schema (can be omited if default schema is used)
