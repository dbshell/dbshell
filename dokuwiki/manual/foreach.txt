====== ForEach element ======

ForEach element repeats contained commands in for each item contained in **Source** property. Current item 
is available in variable, which name is given in **Property** attribute.

  * Implements: **IRunnable**
  * Children must implement: **IRunnable**
  * Source list
  * Attribute **Property**: name of variable, which is filled with current item
