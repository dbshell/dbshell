====== Part 1 – Hello World ======



DB Shell uses XAML language for describing database operations. That means, you write XML, which is transformed objects. Each attribute is mapped to .NET property. Standard elements are classes from DbShell.Core namespace, which can be browsed in [[http://dbshell.com/docs/?topic=html/N_DbShell_Core.htm|online documentation]].

==== Hello World! ====


The following script outputs “Hello World” message.


<code xml>	
<?xml version="1.0" encoding="utf-8" ?>
<Echo
xmlns="http://schemas.dbshell.com/core"
Message="Hello world!"
>
</code>

There is configuration of namespace (xmlns=”http://schemas.dbshell.com/core”). This namespace points to DbShell.Core namespace.
