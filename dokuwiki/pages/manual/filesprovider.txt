====== FilesProvider element ======

Provides list of selected files in directory.

  * Implements: **IListProvider**
  * Attribute **Filter**: file path with filter, eg. "c:/test/*.csv" . In XAML, fore-slash must be used, **back-slash will not work**
  * Extesion syntax: ''Property="{Files 'c:/test/*.csv'}"''
