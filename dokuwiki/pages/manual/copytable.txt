====== CopyTable element ======

CopyTable element copies tabular data. Could be used to following purposes:
  * Export table, view or query result to file
  * Import file to existing table, or create new table from file data
  * Copy one table to another
  * Copy one file to another (and change format)


  * Implements: **IRunnable**
  * Attribute **Source**: data source, must implement **ITabularDataSource**
  * Attribute **SourceExpression**: expression to obtain source data
  * Exactly one of **Source** or **SourceExpression** properties must be defined
  * Attribute **Target**: target, must implement **ITabularDataTarget**
  * Attribute **TargetExpression**: expression to obtain target
  * Exactly one of **Target** or **TargetExpression** properties must be defined
  * Attribute **CleanTarget**: if ''true'', target table/file is cleaned before copying (default: ''false'')
  * Subelement CopyTable.ColumnMap: collection of column mappings (items must implement IColumnMapping), can be omited. 

===== CopyTable.ColumnMap property =====
This property contains list of elements impelementing interface **IColumnMapping**.
It could be one of elements:
==== MapColumn element ====
  * Property **Name** - the name of source column
  * Property **OutputName** - the name of output column. Of omited, **Name** value is used
==== MapValue element ====
  * Property **Name** - the name of output column
  * Property **Value** - value of output column (suitable for text values)
  * Property **Expression** - expression for computing output value
  * Exactly one of properties **Value** and **Expression** must be set
  * Property **NeedColumnValues** - indicates whether column values are available in expressions. By default ''true'', change it to false for performance reasons
