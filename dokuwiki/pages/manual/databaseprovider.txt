====== DatabaseProvider element ======

Provides access to database model. Can be used in [[Razor]] element. 
To available members, see documentation of [[http://dbshell.com/docs/?topic=html/T_DbShell_Driver_Common_Structure_DatabaseInfo.htm|DatabaseInfo class]]

  * Implements: **IModelProvider**
  * Extension syntax: ''Property="{Database}"''
