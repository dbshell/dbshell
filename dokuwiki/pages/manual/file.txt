====== File element ======

Represents file with date,type is determined by extension.

  * Implements: **ITabularDataSource**, **ITabularDataTarget**
  * Attribute **Name**: file name
  * Can represent CDL or CSV file. If you need to specify more details, use specialized elements [[CdlFile]] or [[CsvFile]]
  * Extension syntax: ''Property="{File 'c:/test/file.csv'}"''
