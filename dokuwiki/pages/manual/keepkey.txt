====== KeepKey element ======

Marks primary key of table to be keeped. In target database is inserted the some value as in source database. Default behaviour is to insert rows into related tables and use of identoty values, or insert NULL value, if this is not possible.

  * Attribute **Name**: dataset (by default "DefaultDataSet")
  * Attribute **Table**: source table name
