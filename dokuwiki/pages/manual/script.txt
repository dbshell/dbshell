====== Script element ======

Script element executes script, from external file, or SQL command in **Command** property.

  * Implements: **IRunnable**
  * Attribute **File**: source SQL file name
  * Attribute **Command**: SQL command to be executed
  * Attribute **UseTransactions**: If ''true'', script is executed in transaction (default is ''false'')
  * Attribute **ReplacePattern**: The regular expression, which is used for replacing in scripts. By default, it is @"\$\{([^\}]+)\}"
  * Attribute **UseReplacements**: ''true'' if use replacements using ReplacePattern; otherwise, ''false''. By default, true for inline Command, false for File
