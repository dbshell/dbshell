====== AddRows element ======

Adds rows from table, that meets given condition.

  * Attribute **Name**: dataset (by default "DefaultDataSet")
  * Attribute **Table**: source table name
  * Attribute **Condition**: condition for rows of source table

===== Condition syntax =====
Condition can refer to related tables, use syntax {ForeignKeyColumn}.[ValueColumn]
