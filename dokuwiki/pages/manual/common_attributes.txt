====== Common attributes ======

These attributes are common for more elements in DbShell.

  * Attribute **Connection**: connection string. It not specified, it is inherited from parent element. If not specified in any parent and is run inside [[http://dbmouse.com|DbMouse tool]], current DbMouse connection is used.
  * Attribute **Requires**: Is defined in all elements implementing **IRunnable**. Contains list of included DbShell scripts.
