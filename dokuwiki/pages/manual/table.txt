====== Table element ======

Table element enapsulates table in database. It could be used in following contexts:
  * Data source - Source property of [[CopyTable]] element
  * Data target - Target property of [[CopyTable]] element
  * List of rows - model for [[Razor]] element


  * Implements: **ITabularDataSource**, **ITabularDataTarget**, **IListProvider**, **IEnumerable**, **IModelProvider**
  * Attribute **Name**: table name
  * Attribute **Schema**: table schema (can be omited if default schema is used)
  * Extension syntax: ''Property="{Table Album}"''
