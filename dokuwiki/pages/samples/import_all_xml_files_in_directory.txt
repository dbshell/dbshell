====== Import all XML files in directory ======

This example imports all XML files in "albums" directory into table Album.

<code xml>
<?xml version="1.0" encoding="utf-8" ?>
<Batch
    xmlns="http://schemas.dbshell.com/core"
>
  <ForEach Property="File" Source="{Files 'c:/data/albums/*.xml'}" >
    <Echo Message="Importing file ${File}" />
      <CopyTable Target='{Table Album}'>
        <CopyTable.Source>
          <XmlReader File='${File}' XPath='/Root/Album'>
            <XmlReader.Columns>
              <XmlColumn Name='AlbumId' XPath='@ID' />
              <XmlColumn Name='Name' XPath='Name' />
            </XmlReader.Columns>
          </XmlReader>
        </CopyTable.Source>
     </CopyTable>
  </ForEach>
</Batch>
</code>

Source XML, which is imported by this script, looks like:
<code xml>
<Root>
  <Album AlbumId='1'>
    <Name>Forgotten tales</Name>
  </Album>
</Root>
</code>
