====== Export table and column list to XML ======

This scripts export database structure to simple XML file shown below. We are also settings custom properties to one table and column, these properties are printed in attribute ''Description''.

<code xml>
<?xml version="1.0" encoding="utf-8" ?>
<Batch
    xmlns="http://schemas.dbshell.com/core"
    xmlns:dbmouse="http://schemas.dbmouse.com/dbshell"
>
  <SetTableProperty Name="Comment" Table="Album" Value="List of albums" />
  <SetColumnProperty Name="Comment" Table="Album" Column="AlbumId" Value="Primary key" />
  
  <Razor Model="{Database}" File="c:/test/tables.xml">
  <![CDATA[
        <ROOT>
          @foreach(var table in Model.Tables) {
            <Table Name="@table.Name" Description="@table.Properties["Comment"]" >
              @foreach(var column in table.Columns) {
                <Column Name="@column.Name" Description="@column.Properties["Comment"]" />
              }
            </Table>
          }
        </ROOT>
      ]]>  	
  </Razor>
</Batch>
</code>

Output XML will look like:
<code xml>
<ROOT>
  <Table Name="Album" Description="List of albums" >
    <Column Name="AlbumId" Description="Primary key" />
    <Column Name="Title" Description="" />
    <Column Name="ArtistId" Description="" />
    ...
  </Table>
...
</ROOT>

</code>
