====== Trivial data set ======

This script exports one row from table genre (where GenreId=5)

<code xml>
<?xml version="1.0" encoding="utf-8" ?>
<Batch
    xmlns="http://schemas.dbshell.com/core"
    xmlns:ds="http://schemas.dbshell.com/dataset"
>
	<ds:DataSet />
	<ds:AddRows Table='Genre' Condition="GenreId=5" />
	<ds:Prepare />
	<ds:WriteSql File='ds1.sql' />	
</Batch>
</code>

Output file will look like following:

<code sql>
declare @reg0 int;
create table #memory (id int, value int);
insert into [Genre] ([Name]) values (N'Rock And Roll');
drop table #memory; 
</code>
